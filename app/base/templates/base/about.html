About Issue Tracker

Issue Tracker is a Python web application running on Ubuntu. It uses the Django web framework to clone basic bug tracker features from GitHub and adds the ability to flag issues against folders, files, and individual lines of code. The site is integrated with GitHub's public API. It utilizes the Python client, PyGitHub , and a custom client script to create, update, open, and close GitHub issues. It can also import a GitHub repository folder/file system.
Issue Tracker uses the Django REST framework to consume API data from GitHub. It can import a GitHub repository folder/file structure as well
Issue Tracker runs on an AWS EC2 instance. It stores its data in a Postgres database housed in an RDS instance.
